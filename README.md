# Quản Lý Nhân Sự

Phần mềm quản lý nhân sự được phát triển bằng Java Swing.

## Tính năng

- Quản lý nhân viên
- Quản lý hợp đồng
- Chấm công
- Tuyển dụng
- Thống kê báo cáo

## Cài đặt

1. Clone repository
2. Import database từ file `database_script.sql`
3. Cấu hình kết nối database trong `DatabaseUtils.java`
4. Chạy file `HRM.java`

## Yêu cầu hệ thống

- Java JDK 8+
- SQL Server
